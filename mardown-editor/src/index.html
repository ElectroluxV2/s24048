How to use?

  • Type Markdown or HTML into Input area.
  • You can import code from file.
  • You can export result to file.
  • Scroll down to see the editor.

What is HTML?

  • HTML stands for HyperText Markup Language.
  • It is a standard markup language for web page creation.
  • It allows the creation and structure of sections, paragraphs, and links using HTML elements.
  • This converter supports only subset of HTML.

What is Markdown?

  • Markdown is a lightweight markup language that you can use to add formatting elements to plaintext text documents.
  • Created by John Gruber in 2004, Markdown is now one of the worldâ€™s most popular markup languages.
  • Markdown is platform independent. You can create Markdown-formatted text on any device running any operating system.

Can I use them interchangeably?

  • If you write simple text documents: Yes!
  • If you want to use advanced things like underlines you are stuck with HTML.
